# Employee Management System

A command-line employee management application built with Python. The program allows users to add, search, update, and remove employee records, as well as export the data to a text file for backup.

---

## Purpose

This project was created to strengthen my understanding of:

* Python functions and modular programming
* Lists and dictionaries
* File handling
* Input validation
* Menu-driven application design
* Basic data management

---

## Features

* Add new employee records
* Remove employees by ID
* Update employee salaries
* Search employees by ID or name
* Display all employee records
* Save employee data to a text file for backup

---

## Technologies Used

* Python

---

## How It Works

Employee information is stored in a list of dictionaries during program execution. The application uses a menu-driven interface that allows users to manage employee records and create a backup of the stored data in a text file.

---

## How to Run

Run the program using Python:

```bash
python employee_management.py
```

Follow the on-screen menu to manage employee records.
